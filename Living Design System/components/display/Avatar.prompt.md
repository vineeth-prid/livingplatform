Person or community image with graceful initials fallback and optional status dot.

```jsx
<Avatar name="Anaya Rao" src="/people/anaya.jpg" size="md" status="online" />
<Avatar name="The Meadows" shape="square" size="lg" />
```

Sizes `xs → xl`. Falls back to two-letter initials on a tinted background when `src` is absent.
