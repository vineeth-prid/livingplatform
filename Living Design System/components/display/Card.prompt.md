Calm content surface — the base container for most Living layouts.

```jsx
<Card variant="elevated">Anything</Card>
<Card variant="glass" interactive onClick={open}>Frosted panel over imagery</Card>
```

Variants: `elevated` (soft shadow), `outline` (hairline), `quiet` (sunken fill), `glass` (subtle blur — use only over photography). `padding` `none|sm|md|lg`. Set `interactive` for hover lift.
