# Living — Property Sales Website (UI kit)

A luxury real-estate browsing experience — the public-facing surface of Living.

## Screens
- **Home** (`home.jsx`) — full-bleed hero with glass search bar, stat strip, curated `PropertyCard` grid with filter Tags, ecosystem band, and an image CTA.
- **Property detail** (`detail.jsx`) — editorial gallery mosaic, spec row, description, amenity Tags, and a sticky booking Card with a Living advisor.
- **Chrome** (`chrome.jsx`) — glass-on-scroll `SiteNav` (type-lockup wordmark) and a Pine `SiteFooter`.

## Interaction
`index.html` mounts an interactive app: click any listing → property detail; "Back to homes" returns. Nav becomes a frosted glass bar on scroll.

## Components used
`PropertyCard`, `Button`, `Badge`, `Tag`, `Card`, `Avatar`, `Input` from `window.LivingDesignSystem_bba765`.

## Notes
Photography is Unsplash (architecture/interiors) standing in for real Living imagery — warm, naturally lit, calm compositions per the photography direction.
